# wsl2 설치된 리눅스 배포판 삭제하는 방법

PowerShell을 관리자 권한으로 실행 후, 아래 명령어를 실행합니다.

```powershell
:: 1. 설치된 Linux 배포판 확인
wslconfig.exe /l
:: 2. 설치된 Linux 배포판 삭제
wslconfig.exe /u LinuxName
:: 3. 설치된 Linux 배포판 삭제 확인
wslconfig.exe /l
```

바로 삭제가 됩니다.
